# Cybersecurity Research Hub

A static HTML and CSS quick-reference hub for cybersecurity research, fundamentals, models, networking, role paths, hands-on practice, tools, and trusted external resources.

## Live Site

https://apongpoh.github.io/Cybersecurity-Research/

## What It Covers

- Quick cybersecurity reference terms
- CIA Triad, vulnerability, threat, and risk
- Risk assessment and management flow
- Cyber Kill Chain, Diamond Model, and MITRE ATT&CK thinking
- Network basics, common ports, controls, and packet analysis flow
- Research workflow, evidence handling, validation, and reporting
- CVE, CVSS, CWE, CPE, NVD, disclosure, remediation, and mitigation
- Cybersecurity role paths
- Labs, certifications, Kali Linux tools, and official references

## Files

- `index.html` - main page content and links
- `styles.css` - responsive styling and layout

## How To View

Open `index.html` directly in a browser.

No build step or server is required.

## Advice

Stay curious, verify every claim, document your evidence, and use these references only on systems you own or are authorized to test.
